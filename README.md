# Abyss Strider 深渊行者

🎮 一款万圣节主题的Roguelike动作游戏

## 🎯 游戏特色

- **三种武器流派**: 法杖、长剑、双刀，各具特色
- **构筑系统**: 丰富的构筑卡牌，打造独特build
- **5个Boss关卡**: 险恶猴子、冰龙、地狱三头犬、雷神宙斯、神圣骑士
- **赐福系统**: 击杀敌人获得强力赐福
- **武器升级**: 6级武器进化，解锁强力技能

## 🎮 操作说明

| 操作 | 按键 |
|------|------|
| 移动 | WASD / 方向键 |
| 切换武器 | Q |
| 暂停 | ESC / P |

## 💻 本地运行

由于使用ES6模块，需要本地服务器运行：

```bash
# 使用Python
python -m http.server 8000

# 或使用Node.js
npx serve
```

然后访问 `http://localhost:8000`

## 🛠️ 技术栈

- 原生 JavaScript (ES6 Modules)
- HTML5 Canvas
- CSS3 动画
