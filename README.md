# Abyss Strider 深渊行者

🎮 一款万圣节主题的Roguelike动作游戏 | **v1.0.6**

## 🎯 游戏特色

- **三种武器流派**: 法杖、长剑、双刀，各具特色
- **构筑系统**: 丰富的构筑卡牌，打造独特build
- **5个Boss关卡**: 险恶猴王、冰霜巨龙、地狱三头魔犬、天穹之王·宙斯、圣剑王·亚瑟
- **赐福系统**: 击杀敌人获得强力赐福
- **武器升级**: 6级武器进化，解锁强力技能

## 💻 本地运行

由于使用ES6模块，需要本地服务器运行：

```bash
# 使用Python
python -m http.server 8000

# 或使用Node.js
npx serve
```

然后访问 `http://localhost:8000`
